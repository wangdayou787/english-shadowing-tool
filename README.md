# English Shadowing Tool

A personalized English shadowing and learning tool powered by AI.

## Features
- **AI Text Generation**: Generates educational content based on grade and interest.
- **Audio Shadowing**: Standard pronunciation using Edge TTS or Aliyun Qwen TTS.
- **Pronunciation Assessment**: Evaluate your recording against the text.
- **Vocabulary & Grammar Analysis**: Auto-generated learning points.

## Deployment
Deployed on Streamlit Cloud.
